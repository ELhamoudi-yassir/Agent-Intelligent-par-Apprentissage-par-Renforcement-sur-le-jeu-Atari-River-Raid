# 🎮 Agent Intelligent par Apprentissage par Renforcement — Atari River Raid

Agent IA autonome entraîné à jouer au jeu Atari River Raid via Deep 
Reinforcement Learning (PPO + CNN), développé au Laboratoire XLIM, 
Université de Limoges.

## 🤖 Description

Conception et implémentation d'un agent de Deep RL capable d'apprendre 
à jouer à River Raid de manière autonome, atteignant un taux de réussite 
de 70% après 10 épisodes d'entraînement.

## 🧠 Architecture

- **Algorithme** : PPO (Proximal Policy Optimization)
- **Réseau de neurones** : CNN (Convolutional Neural Network)
- **Environnement** : OpenAI Gym Retro
- **Framework** : PyTorch

## 📊 Résultats

- Taux de réussite : **70%** après 10 épisodes
- Optimisation des fonctions de récompense et hyperparamètres
- Évaluation sur environnements complexes multi-niveaux

## 🚀 Lancer le projet

```bash
pip install torch gym-retro
python train.py
```
